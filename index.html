git config --global user.name - имя автора данного коммита git config --global user.email - почта, куда будет приходить инфа о коммитах с которыми работаем git init - создает новый репозиторий нашего файла, запускает git status - проверяет статус текущей работы, что мы там надели git add - добавление новых текущих изминений, файлов git commit - данные которые находятся в индексе, переносятся в репозиторий git log - истоория коммитов именно в даной ветки git checkout - выбирает данную ветку и переключается на ее, обновляя до робочего состояние git reset - если увидели ошибку, можно ее отменить, с помощью этой команды
